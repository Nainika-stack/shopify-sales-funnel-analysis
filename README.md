# Shopify Sales Funnel Analysis

## Overview
This project analyzes customer purchasing behavior and sales funnel performance using retail transaction data.

## Objectives
- Identify customer drop-off points
- Analyze conversion rates
- Improve customer retention insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Tableau

## Dataset
Retail transaction dataset containing customer purchases and product details.

## Key Insights
- Highest customer drop-off occurs at checkout stage
- Returning customers contribute 62% of total revenue
- Seasonal sales spikes observed during Q4


## Conclusion
The analysis helps businesses optimize sales funnel performance and customer engagement strategies.
